Risk-Adjusted Portfolio Optimization and Stability Analysis

Project Overview

Comprehensive quantitative analysis comparing multiple portfolio optimization strategies using 10+ years of historical ETF data (2015-2026). This project demonstrates advanced optimization techniques and rigorous out-of-sample testing methodology.

Key Accomplishments

  • Analyzed 6 Diversified ETF Assets: SPY (S&P 500), QQQ (NASDAQ), EEM (Emerging Markets), VNQ (Real Estate), GLD (Gold), SLV (Silver)
  • Implemented 5 Advanced Optimization Strategies: Max Sharpe Ratio, Ledoit-Wolf Shrinkage, Minimum Variance, Risk Parity, and Constrained Optimization with Position Limits
  • Dynamic Rebalancing: Rolling window optimization with quarterly rebalancing to adapt weights based on recent market conditions
  • Robust Validation: 500 bootstrap resampling iterations to confirm portfolio stability across different market regimes
  • Out-of-Sample Testing: Train-test split (2015-2020 training, 2020-2025 testing) prevents overfitting and ensures realistic performance estimates
  • Comprehensive Metrics: Evaluated all strategies using Annual Return, Volatility, Sharpe Ratio, and Maximum Drawdown metrics

Technical Implementation

  • Data Source: Yahoo Finance (via yfinance Python library)
  • Optimization Method: Sequential Least Squares Programming (SLSQP) algorithm
  • Statistical Methods: Covariance estimation, shrinkage estimators (Ledoit-Wolf), correlation analysis
  • Technologies: Python, pandas, numpy, scipy, scikit-learn, matplotlib
  • Analysis Period: 10+ years of daily market data

Key Findings

  • Optimization Improves Returns: All optimized strategies significantly outperformed the equal-weight baseline portfolio
  • Shrinkage Estimation Works: Ledoit-Wolf covariance estimator reduced estimation error and improved out-of-sample performance
  • Risk Management Benefits: Constraining positions to 30% maximum improved risk-adjusted returns by reducing concentration risk
  • Dynamic Outperforms Static: Rolling window rebalancing with quarterly updates captured changing market conditions and correlations
  • Commodity Diversification: Gold and silver allocations provided portfolio stabilization during equity market volatility
  • Bootstrap Validation: Portfolio weights showed stability across 500 resampling iterations, confirming robustness
  • Out-of-Sample Validation Critical: This analysis demonstrated why out-of-sample testing is essential for avoiding overfitting bias

Optimization Strategies Compared

1. Original (Max Sharpe Ratio)

Maximizes risk-adjusted returns using standard portfolio optimization. Baseline approach that serves as foundation for comparison.

2. Ledoit-Wolf Shrinkage Estimator

Improves covariance matrix estimation through statistical shrinkage. Reduces estimation error in small-sample scenarios and stabilizes predictions.

3. Minimum Variance Portfolio

Minimizes portfolio volatility regardless of expected returns. Often outperforms in practice due to robustness against mean-return estimation errors.

4. Risk Parity Portfolio

Allocates weights to achieve equal risk contribution from each asset. Simple, intuitive approach with natural diversification properties.

5. Constrained Optimization (30% Position Limit)

Maximizes Sharpe ratio while limiting individual positions to 30%. Prevents excessive concentration and improves practical implementability.

6. Rolling Window with Quarterly Rebalancing

Adapts portfolio weights quarterly based on recent market conditions. Uses 1-year rolling lookback window to capture tactical allocation opportunities.

Performance Metrics

  • Annual Return: Average yearly returns over out-of-sample test period
  • Volatility (Annualized): Standard deviation of daily returns scaled to annual basis
  • Sharpe Ratio: Risk-adjusted return metric; higher indicates better risk-adjusted performance
  • Maximum Drawdown: Largest peak-to-trough decline; indicates worst-case portfolio loss
